Uitvoering van een taak. Bekende typen hebben een variant met een `const`-waarde op
`type` en type-specifieke velden. `OnbekendeUitvoering` dient als catch-all voor
toekomstige of onbekende typen en zorgt ervoor dat `oneOf` altijd precies één match
oplevert.

Discriminator: veld `type`.
