Samengesteld object met klantgerelateerde gegevens.
Bevat per resource-type een lijst van **samenvattingen**. Voor resource-typen
waarvoor een detail-endpoint bestaat (bijv. `GET /taken/{uuid}` voor afhandeling)
kan aanvullende informatie worden opgevraagd.
Bevat alleen de resource-typen die via `include` zijn opgevraagd
én door de provider worden ondersteund. Afwezige sleutels betekenen
dat het type niet is opgevraagd of niet ondersteund wordt.
