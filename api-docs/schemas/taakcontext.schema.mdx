De context waaraan de taak is gekoppeld (bijv. een zaak of product).

De context wordt aangeduid met een **URN** (portaal-agnostische identifier)
en optioneel een **canonicalUrl** (door de dienstverlener aangeleverd).

`canonicalUrl` is afwezig als de dienstverlener geen eigen portaal heeft
— bijvoorbeeld gemeenten die volledig via MijnOverheid werken.

Portalen met contextondersteuning kunnen een interne URL opbouwen via de
URN; het type context is af te leiden uit de URN-structuur zelf.

| Kan URN omzetten | `canonicalUrl` aanwezig | Actie |
|---|---|---|
| ja | ja/nee | Bouw interne URL via URN; `canonicalUrl` irrelevant |
| nee | ja | Gebruik `canonicalUrl` als directe link |
| nee | nee | _(nader te bepalen)_ |
