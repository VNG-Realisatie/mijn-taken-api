Declaratieve definitie voor lokale uitvoering van een taak. Alleen
aanwezig in de detailrespons (`GET /taken/{uuid}`).

Betaling heeft geen `definitie` — betaaltaken leiden altijd
tot een redirect via `uitvoering.canonicalUrl`.

Welke velden aanwezig zijn volgt uit `uitvoering.type`. Clients die
een onbekend type tegenkomen negeren `definitie` en gebruiken
`canonicalUrl` als fallback.
